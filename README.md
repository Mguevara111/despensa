# 🍷 Virtual Market - Gourmet Store Landing Page

[English](#english) | [Español](#español)

---

## English

### 🚀 Description
A professional landing page for a gourmet store specializing in cold cuts, cheeses, and wines. This project showcases a clean, commercial UI with a fully functional integration of the **PayPal Sandbox API** for secure payment processing.

### ✨ Key Features
* **E-commerce Ready:** Integration with PayPal for real-time transaction testing.
* **Responsive Design:** Fully adapted for mobile and desktop devices using Vanilla CSS.
* **Shopping Cart Logic:** Custom JavaScript to manage products, totals, and validation before checkout.
* **Deployment:** Hosted on **Netlify** with continuous integration.

### 🛠️ Technologies
* **Languages:** HTML5, CSS3, JavaScript (ES6+).
* **Payment Gateway:** PayPal SDK (Sandbox Mode).
* **Styling:** Custom Maquetación (Flexbox/Grid).

## Español

### 🚀 Descripción
Una landing page profesional para una tienda gourmet de embutidos, quesos y vinos. Este proyecto destaca por una interfaz comercial limpia y una integración funcional con la **API de PayPal Sandbox** para el procesamiento de pagos seguros.

### ✨ Funcionalidades Clave
* **E-commerce Ready:** Integración con PayPal para pruebas de transacciones en tiempo real.
* **Diseño Responsivo:** Completamente adaptado a dispositivos móviles y de escritorio usando CSS puro.
* **Lógica de Carrito:** JavaScript personalizado para gestionar productos, totales y validación previa al pago.
* **Despliegue:** Alojado en **Netlify** con integración continua.

### 🛠️ Tecnologías utilizadas
* **Lenguajes:** HTML5, CSS3, JavaScript (ES6+).
* **Pasarela de Pagos:** SDK de PayPal (Modo Sandbox).
* **Maquetación:** CSS Personalizado (Flexbox/Grid).
